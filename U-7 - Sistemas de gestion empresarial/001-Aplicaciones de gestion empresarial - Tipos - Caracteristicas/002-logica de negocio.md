Lógica de negocio
Reglas que gobiernan el comportamiento de ese negocio
"Como funciona esa empresa"
"Como fluye la información"

Si en una empresa la lógica de negocio está definida
Se puede hacer un software para automatizar esas reglas

Si en una empresa la lógica de negocio no está bien definida
No se puede hacer un software
Primero hay que ordenar la lógica de negocio
Y luego construir un software

Vamos a simular que tenemos que establecer una lógica de negocio
Cada uno de vosotros elige un tipo de negocio
Recomendación: Coged una lógica de negocio de la cual conozcáis algo 
(o preguntad si no lo conocéis)

Por ejemplo: Yo conozco centros de formación
Lógica de negocio:

1.-LLega una persona interesada
2.-Realiza una inscripción
3.-Secretaría valida la inscripción
4.-Con la inscripción validada, se convierte en matrícula
5.-Qué es una matrícula?
	FP - Ciclos, Modalidades, Cursos, Módulos profesionales
	Master - Como mucho tiene asignaturas
	Cursos cortos
	...
6.-Generalmente un alumno necesita un acceso a una plataforma (LMS) - Crear usuario MS
7.-Se crea un registro en un SIS (notas, asistencias, etc)
8.-Si el alumno realiza, pagos, la información debe viajar hasta factura/contabilidad.
